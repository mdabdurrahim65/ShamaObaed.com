# Shama Obaed Official Website

This repository contains the **official political profile website of Shama Obaed**, a BNP Organising Secretary and candidate from Faridpur-2 constituency in Bangladesh.  

The website is designed to be **clean, professional, responsive, and mobile-friendly**, with multiple sections to showcase her biography, vision, policies, statements, gallery, and contact information.  

## Features

- **Hero Section Slider**: Multiple images with tagline for professional presentation  
- **About Section**: Biography with profile image  
- **Vision & Policies Section**: Cards highlighting key political goals and social initiatives  
- **Statements Section**: Carousel of important public statements  
- **Gallery Section**: Horizontal scroll of images  
- **Contact & CTA**: Facebook link for direct connection  
- **SEO Optimized**: Meta tags, description, keywords, and Open Graph tags included  
- **Ads Integration**: 10 advertisement slots integrated using JavaScript  
- **Responsive Design**: Works seamlessly on mobile, tablet, and desktop

4. Your website will be live at `https://<username>.github.io/<repository>`
